# QB1 Peripherals

This document lists peripherals that are supported by QB1. This list is not comprehensive, and adding support for a 3.3v sensor is usually a simple matter of developing a face plate for it. 

## Sensors

### Distance

* [HC-SR04](HCSR04). An ultrasonic distance sensor with a range of 2cm to 4m and an accuracy of 3mm. Interfaces to QB1 using a peripheral board. 
* [VL53LOX](https://makerworld.com/en/models/2572612-qb1-vl53lox-plate). A time-of-flight laser distance sensor with a range of 30mm to 2m and an accuracy of +/- 3%. Connects directly to QB1 using wires. 

### Motion/Presence

* [AM312](https://makerworld.com/en/models/2547169-qb1-am312-plate). A PIR motion sensor that detects humans and pets up to 5m away. Connects directly to QB1 using wires. An [angled mounting plate](https://makerworld.com/en/models/2548871-qb1-am312-plate-angled) is also available.

### Environmental

* [SHT40](https://makerworld.com/en/models/2571182-qb1-sht40-plate). A temperature and humidity sensor with an accuracy of +/- 0.2 degrees C and +/- 1.8% RH. Connects directly to QB1 using wires. 

### Vibration  

* [SW-420](https://makerworld.com/en/models/2572635-qb1-sw-420-plate) vibration sensor. Connects directly to QB1 using wires.
