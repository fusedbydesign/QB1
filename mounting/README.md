# QB1 Mounting

This document lists QB1 mounting hardware and other physical solutions.

## Mounting

* A face plate is available that allows the QB1 to be [mounted to a wall or ceiling](https://makerworld.com/en/models/2566908-qb1-wall-ceiling-mount-plate) using screws.

## Miscellaneous

* QB1 faces that are not used for peripherals or mounting can be covered with [blank face plates](https://makerworld.com/en/models/2543181-qb1-blank-face-plate).
